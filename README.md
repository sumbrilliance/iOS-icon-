# iOS-icon-
logo生成器，只需要美工同学提供1024*1024的原图（因为这个图你上架迟早要用），其余icon一次生成，省时省力。
默认生成图片以pt和2倍、3倍图作为新图后缀，生成目录与原图目录相同。
例如，
>logo.png   // 原图1024*1024
生成
>logo29@2x.png // 58*58
>logo29@3x.png // 87*87
>logo40@2x.png // 80*80
>logo40@3x.png // 120*120
>logo60@2x.png // 120*120
>logo60@3x.png // 120*120
